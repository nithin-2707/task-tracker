# Task Tracker

A simple and clean web application for managing daily tasks.

## Features

- Add new tasks
- View all tasks in a list
- Mark tasks as completed or pending
- Track task status with visual indicators

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Clone this repository or download the files
2. Open `index.html` in your web browser
3. Start adding and managing your tasks

## Project Structure

```
task-tracker/
├── index.html      # Main HTML file
├── styles.css      # Styling
├── script.js       # JavaScript functionality
├── logo1.png       # Logo image
├── favi.png        # Favicon
└── README.md       # Documentation
```

## Usage

1. Type your task in the input field
2. Click "Add Task" or press Enter
3. Click the checkmark (✓) to mark as completed
4. Click the refresh icon (↺) to mark as pending again
